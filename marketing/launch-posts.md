## Post 4 — The AI Angle (HackerNews / LinkedIn)

**Show HN: I built an AI voice agent that cold-calls Sydney cafes to ask the price of a flat white**

Stack: Node.js, Twilio + OpenAI Realtime, Supabase, Leaflet.

The agent (Mia) calls each cafe, confirms the name, asks the price, confirms it, hangs up. 30 seconds per call. 55% answer rate.

We've called 500+ cafes so far. Every price verified by phone, not scraped from menus.

Live dashboard with map, suburb rankings, and price comparison tools.

## Post 5 — The Methodology (Twitter thread / LinkedIn)

How do you price 2,000 cafes?

1. Discover every cafe in Sydney via Google Places API
2. Filter out chains (Starbucks, Gloria Jeans, etc.)
3. Have an AI voice agent call each one during business hours
4. Ask one question: "How much is a regular flat white?"
5. Extract the price from the conversation
6. Plot it on a live dashboard

We've done 500 so far. The map is filling up.

→ flatwhiteindex.com.au
