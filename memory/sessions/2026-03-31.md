## Key Learnings
- process.exit in deploy endpoint kills connection before response flushes
- Leaflet CSS preload onload fails on Firefox Android
- Supabase FK join breaks after migration recreates FK (needs schema reload)
- Voice prompt needs explicit STOP and WAIT between steps
- X-Frame-Options blocks embedding — use CSP frame-ancestors instead
