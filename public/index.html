How this data is collected

Who collects the prices?
An AI voice agent named Mia rings each independent cafe and asks one question: “How much is a regular flat white?” The price is extracted from the call transcript and stored. Chain cafes are excluded — only independents are surveyed. Calls run 9am–4pm AEST, weekdays only.
Can I use the data?
Yes — the full dataset is open under CC BY 4.0: JSON export · llms.txt for AI. Attribute as “Source: Flat White Index, flatwhiteindex.com.au”.
How fresh are the numbers?
Every figure on this page is computed live from the dataset at the moment you load it. Sample size is shown for every suburb in the league table. Read the full methodology →
